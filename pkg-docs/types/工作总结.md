## 模块概述

Types 模块（`@alilc/lowcode-types`）是低代码引擎的类型定义核心模块，包含：

- **239 个源文件**：TypeScript 类型定义文件
- **500+ 个类型定义**：接口、类型别名、枚举等
- **20+ 个枚举**：各种枚举类型
- **300+ 个接口**：核心接口定义
- **200+ 个类型别名**：类型别名定义

### 核心功能模块
1. **Editor 类型** - 编辑器核心类型
2. **Activity 类型** - 活动类型
3. **Code 类型** - 代码中间表示和结果类型
4. **Assets 类型** - 资源类型
5. **Event 类型** - 事件系统类型
6. **Shell API 类型** - 13 个 API 子模块
7. **Shell Model 类型** - 30 个模型类型
8. **Shell Type 定义** - 100+ 个类型定义文件
9. **Shell Enum 类型** - 7 个枚举类型
10. **Deprecated 类型** - 17 个已弃用类型
